⚙ 設定 - Google Drive 同期

別デバイスとデータを同期するには、Google Cloud で OAuth Client ID を作成して下記に貼り付けます。
📖 セットアップ手順を見る

  1. Google Cloud Console でプロジェクト作成
  2. 「APIとサービス」→「ライブラリ」→ Google Drive API を有効化
  3. 「OAuth同意画面」を構成（外部・自分のメールを追加）
  4. 「認証情報」→ OAuth クライアントID 作成（種類：ウェブアプリ）
  5. 「承認済みの JavaScript 生成元」にこのアプリのURLを追加
  6. 発行された Client ID（〜.apps.googleusercontent.com）を下にコピペ
